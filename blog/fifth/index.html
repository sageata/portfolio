Better Forms

iSurvey

Before you ask, this project has not been released in to the world, however I can provide you with demo iOS or Android app.

We all heard about Survey Monkey, Qualtrics, and Microsoft Forms. They are doing the same thing, better or worse, forms. Each has it’s own strengths but also glaring weaknesses, for example Microsoft Forms is great because is integrated with al Microsoft Power Suite, can trigger flows, approvals, and by default is secured behind authentication, however it lacks any sort of advanced features such as limitings the number of characters, or requesting an answer in a specific data type. MS Forms is good only for very basic surveys.

Survey Monkey is very expansive but it provides the best in class customisation for the questionnaire. However, their forms creation experience is overly complicated and it feels that one needs a PHD to create a simple branching.

This is iSurvey came in, the project aims in aiding the author of the questionnaire in getting the data the way they expect and not leaving this at the mercy of the person feeling the form. For example it allows collection of Date fields, Date and Time, and DateTime fields that are formatted as per the author.

The app also helps the form users filling the correct date time with the simplicity of pressing a button for both local time and UTC.

For this project I’ve used Firebase as the backend platform and Flutter/Dart targeting all major platforms, Web, iOS, Android and Windows using the same code base.

The app comes with few other neat features such as the ability to share the survey with other authors, copy survey results to clipboard, and duplicate questions.
