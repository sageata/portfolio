Onboard Cabin Reports - Power App

In December 2021, the Onboard Cabin Report, previously mentioned, got a huge update, the categories have been updated to a 5 level hierarchical taxonomy. These categories have been built in the OCR form and now the crew are able to pre-classify the OCR making the ML model obsolete, however the consolidation issue remained. Also, reading, actioning, and keeping track of the OCRs from Excel was a huge pain point due to the unfriendly interface and lack of images and email integration

Aiming to solve this, I have create a Power Automate Flow that consolidated all the OCR in a SharePoint List and on top of the list I have built a Power App that provides the users with a much better experience than reading them from Excel. The app also allows the users to correct the classification, send emails, view images, and update status.

As we have been able to centralise the OCR data processing

Key outcomes

  • the majority of the OCRs are now actioned from within this app
  • enabled collaboration between departments
  • enabled us to have an overview of the OCRs that have been actioned and which have not
  • created Power BI dashboard that automatically updates instead of creating reports from scratch
  • closed loop on compliance with audited process requirements
