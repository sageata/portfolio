Six Sigma - Part One

Abusing Excel

End of 2022, I have been tasked to prepare the forms for Six Sigma time study. After looking at previous similar work, done by my peers, I have realised that it’s a mad man’s job. Previously the data has been collected using pen and paper and then manually inputed into an Excel file.

There were three main things that the observer needed to note, for each action, and there were hundreads of them:

  • the time of the action
  • the action type
  • the action subtype

Although I was excited about doing the data analysis in Power BI, I was equally frustrated that I could not find a decent tool for collecting the data we needed, therefore I have created an Excel file that would help us collect the time stamps.

I’m not going to bore you with all the technical aspects of the file but it worked like this:

  • there are 2 columns with data validated values of 1000 rows each
  • users could select one of the predetermined actions for each row
  • once they have selected an action two things would happen:
    • a time stamp would be added on a different column
    • the cell next to it had built in validation that was filtered to show only sub-actions for the selected action
  • once the observer selected the action and sub-action, he/she could also assign a seat number for the action
  • the seats were filtered based on the Aircraft Side selection

The file has been opened iPads and we used these without internet, using the ‘Use Offline’ mode.

The observers had a short 30-60 minutes training session, where I mostly rambled about having 100% battery and how to open the file, but the actual usage of the file was so simple that they all understood in seconds on how it works.

Using this file we have been able to do accurate studies on over 50 flights and received amazing feedback from the users that have been doing this using pen and paper before. More importantly, we been able to start the data analysis right away, saving precious time and getting valuable insights with clockwork precision.
