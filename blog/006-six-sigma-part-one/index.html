Six Sigma - Part One

Abusing Excel: Streamlining Six Sigma Time Studies"

In late 2022, I was tasked with preparing forms for Six Sigma time studies. Traditional methods involved pen and paper, followed by manual data entry into Excel—a laborious and error-prone process. I decided to leverage Excel’s capabilities to create a more efficient data collection tool.

The Challenge

The observer had to note three key details for each action, and there could be hundreds of such actions:

  • Time of the action
  • Action type
  • Action subtype

The Solution: An Excel Powerhouse

Unable to find a suitable tool for data collection, I created an Excel file designed to streamline the process. Here’s how it worked:

  • Two columns with data-validated values, each containing 1000 rows.
  • Users could select a predetermined action for each row.
  • Upon selecting an action, a timestamp would be generated, and a filtered list of sub-actions would appear for selection.
  • Observers could also assign a seat number, which was filtered based on the Aircraft Side selection.

User Training and Adoption

The file was deployed on iPads in ‘Use Offline’ mode. A brief 30-60 minute training session was all it took for the observers to get the hang of it, thanks to the file’s intuitive design.

Key Outcomes

  • Efficiency: Conducted accurate studies on over 50 flights.
  • User Satisfaction: Received positive feedback from users who had previously relied on pen and paper.
  • Immediate Data Analysis: Enabled real-time data analysis, saving time and providing valuable insights with precision.
