Power Platform

In December 2021, the Onboard Cabin Report (OCR) underwent a significant update. The categories were expanded to a five-level hierarchical taxonomy, allowing crew members to pre-classify the OCRs. While this made the machine learning model I had previously developed obsolete, the issue of data consolidation remained. Moreover, the Excel interface was cumbersome and lacked features like image and email integration.

The Challenge

The new taxonomy improved classification but didn’t address the core issues of data consolidation and user experience. Reading, actioning, and tracking OCRs from Excel was still a pain point.

The Solution

To tackle these challenges, I created a Power Automate Flow that consolidated all OCRs into a SharePoint List. I then built a Power App on top of this list to provide a more user-friendly experience. The app allows users to correct classifications, send emails, view images, and update statuses, all within a single interface.

The Innovation

The real breakthrough came when we centralized OCR data processing. This centralization enabled real-time updates and facilitated inter-departmental collaboration.

Key Outcomes

  • Streamlined Actioning: The majority of OCRs are now actioned directly within the app, improving efficiency.
  • Enhanced Collaboration: Enabled seamless collaboration between different departments.
  • Improved Oversight: Provided a comprehensive overview of actioned and pending OCRs.
  • Automated Reporting: Created a Power BI dashboard that updates automatically, eliminating the need to build reports from scratch.
  • Compliance Assurance: Closed the loop on compliance by meeting audited process requirements.
